# Cart-Abandonment Dashboard – Pure HTML & CSS

A clean, ready-to-use dashboard that shows live cart-abandonment data.  
No JavaScript, no frameworks—just one HTML file and one CSS file.  
Open `index.html` in any browser and you’re done.

---

## What you get
- KPI cards: abandoned carts, recovery rate, potential revenue, emails sent  
- Searchable list of every abandoned cart with customer, product, value, device, location and status  
- “Most abandoned products” sidebar with emoji icons  
- Device breakdown bars (desktop / mobile)  
- Action buttons: launch campaign, view report, download analytics, settings
